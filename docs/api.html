* CHECK NICK NAME *

<Kiểm tra nickname đã tồn tại hay chưa.>

  • IdCommand

    CHECK_NICK_NAME:int = 1

  • Input:

    username:string (username cần check)

  • Output:

    status:int (1=thành công, 0=thất bại)

* AUTO JOIN ROOM *

<User nhấn tham gia, server tự đẩy vào 1 room phù hợp.>

  • IdCommand

    AUTO_JOIN_ROOM:int = 2

  • Input:

    username:string

  • Output:

    status:int (1=thành công, 0=thất bại)

    roomid:int (roomid hiện tại user đang vào)

    playerinfo:object (thông tin của player trong room)

* GET ROOM INFO *

<Trả về thông tin room mà user đang tham gia.>

  • IdCommand

    GET_ROOM_INFO:int = 3

  • Input:

    roomid:int

  • Output:

    status:int (1=thành công, 0=thất bại)

    playerlist:[ playersteam1:[Object Player], playersteam2:[] ] (playerlist chứa 2 mảng player hiện tại của 2 team)

    roomname:string (tên phòng)

    totalplayer:int (tổng số user đang trong phòng)
